# Go Port Scan

A simple and fast SYN port scanner implemented in GO.

## Building/Compiling EXE

```CMD
go build GoPortScan.go
```

## Running

```CMD
GoPortScan.exe -targetIP=127.0.0.1 -startPort=20 -endPort=65000
```
